Cán bộ hướng dẫn: ThS. Đoàn Văn Trung

Sinh viên thực hiện: Nguyễn Vũ Chí Tình

Khoá: K14

Đề tài: Xây dựng ứng dụng di động quản lý và đặt vé rạp chiếu phim NVC.
